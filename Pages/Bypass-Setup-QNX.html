HƯỚNG DẪN BLACKBERRY 10 PLAYBOOK FIRMWARE

Hạ Cấp BlackBerry 10 Xuống Bản Thấp Hơn 10.3.0 Và Vượt Initial Setup Cho PlayBook / Làm Debloated ROM 10.3.0 Trở Xuống

04/05/2026
Cảnh Báo: Hướng dẫn này liên quan đến việc chỉnh sửa trực tiếp vào firmware của thiết bị. Dành cho người dùng nâng cao. Thực hiện sai bước có thể brick máy vĩnh viễn. Tác giả không chịu trách nhiệm về bất kỳ thiệt hại nào xảy ra với thiết bị của bạn.
Kể từ khi BlackBerry tắt toàn bộ server vào năm 2022, các thiết bị BB10 và PlayBook sau khi factory reset hoặc security wipe sẽ bị kẹt ở màn hình cài đặt ban đầu (initial setup) và không thể tiếp tục. Hướng dẫn này giúp bạn vá firmware để bỏ qua hoàn toàn màn hình đó, đồng thời áp dụng cho việc hạ cấp OS xuống thấp hơn 10.3.0 và làm debloated ROM.
Mục Lục
  1. Thiết bị đã được kiểm tra
  2. Yêu cầu chuẩn bị
  3. Bước 1: Tách file Autoloader
  4. Bước 2: Giải nén file OS (QCFM)
  5. Bước 3: Chỉnh sửa file cấu hình
  6. Bước 4: Đóng gói lại file QCFM
  7. Bước 5: Tạo Autoloader mới
  8. Flash vào thiết bị

Thiết Bị Đã Được Kiểm Tra

  • BlackBerry PlayBook 3G/HSPA+ – OS 2.1.0.1917
  • BlackBerry Z30 – OS 10.3.3.3216 hạ cấp xuống 10.2.1.1057, bị kẹt ở màn hình setup (đã vượt qua bằng hướng dẫn này)

Yêu Cầu Chuẩn Bị

Trước khi bắt đầu, hãy chắc chắn bạn đã có đầy đủ các công cụ sau (đa số đã đều ở trong mục 10.3.0 Trở Xuống của BlackBerry 10 Tools):

  • Cáp USB chất lượng tốt
  • 2 File Autoloader (.exe) đúng với model thiết bị của bạn. 1 cái là bản bạn muốn xuống 10.2, 10.1, 10.0 và bản GỐC 10.3.3.3216 để hạ cấp.
  • DBBT (Darcy's BlackBerry Tools) – công cụ extract file từ BlackBerry Autoloader
  • bb10mt – BlackBerry 10 MultiTool (dùng để unpack/repack QCFM)
  • Hex Editor
  • DBBT (Darcy's BlackBerry Tools) - Modified – Chỉnh sửa để load file QCFM chưa ký và tạo Autoloader cuối cùng
    • QUAN TRỌNG NHẤT
  • Bạn phải xóa OS Block, nếu bạn chưa hiểu cách xóa OS Block, hãy tự tạo&chạy ROM Modified tạo bằng BB10 Tools sử dụng bản rom mới nhất 10.3.3.3216, rồi xem bước 3 tại trang này BBID-PATCH

    • Bạn có thể tải bộ công cụ BlackBerry 10 Tools tại đây:

    BlackBerry 10 Tools
Yêu Cầu Kỹ Năng: Bạn cần quen với việc dùng Command Prompt hoặc PowerShell. Cần biết các lệnh cơ bản như cd, dir và cách điều hướng thư mục.
1

Tách File Autoloader

Đầu tiên, bạn cần extract các file OS và Radio từ file Autoloader chính.

  1. Đặt file Autoloader .exe vào một thư mục mới.
  2. Mở DBBT, chuyển sang tab Extract.
  3. Nhấn Browse và chọn file Autoloader của bạn và bấm tiếp Split Autoloader
  4. DBBT sẽ xử lý và xuất ra hai file .signed trong cùng thư mục với Autoloader:
    • File OS (ví dụ: filename@...OS.signed)
    • File Radio (ví dụ: ...Radio.signed)
  5. Để dễ làm việc, hãy đổi tên chúng thành OS.signed và Radio.signed.
2

Giải Nén File OS (Unpack QCFM)

File .signed chứa OS là một archive QCFM. Dùng bb10mt để giải nén nó. Tuy nhiên, để dễ dàng làm việc và bạn có thể copy phần ví dụ, hãy tạo thư mục BlackBerry trong ổ C:\ và mang bb10mt, 2 file .signed đến thư mục đó.

  1. Mở Command Prompt hoặc PowerShell trong thư mục chứa bb10mt.exe.
    Mẹo: Giữ Shift + chuột phải vào thư mục → chọn "Open PowerShell window here".
  2. Chạy lệnh sau:
bb10mt.exe unpack -c "đường_dẫn\đến\OS.signed"

Ví dụ:

bb10mt.exe unpack -c C:\BlackBerry\OS.signed
Mẹo: Trên Windows Terminal, bạn có thể kéo thả file trực tiếp vào cửa sổ terminal để điền đường dẫn tự động.

Sau khi unpack, bạn sẽ có một thư mục chứa nhiều file. Chúng ta chỉ cần file .ufs.

3

Chỉnh Sửa File Cấu Hình (Quan Trọng Nhất)

Cẩn Thận: Đây là bước then chốt. Hãy thực hiện cẩn thận để tránh phải lặp lại toàn bộ quy trình.
  1. Mở Hex Editor (ví dụ HxD).
  2. Mở file .ufs vừa unpack (kéo thả vào Hex Editor).
  3. Nhấn Ctrl+F để tìm kiếm, nhập chuỗi văn bản: autorun::1

Dành cho thiết bị BlackBerry 10:

Tìm autorun::1 và đổi giá trị 1 thành 0:

autorun::1  →  autorun::0

Dành cho BlackBerry PlayBook:

Tìm và sửa cả hai giá trị sau (lưu ý có 2 entry trùng lặp, phải sửa hết):

deactivate_mode::1  →  deactivate_mode::0
autorun::1          →  autorun::0
Lưu Ý PlayBook: Có hai entry autorun::1 trùng lặp trong file. Bạn phải tìm và đổi cả hai, không chỉ một.
  1. Lưu file sau khi chỉnh sửa (Ctrl+S) rồi đóng Hex Editor.
4

Đóng Gói Lại File QCFM (Pack)

Sau khi đã vá file, cần đóng gói lại thành file .signed mới. Để cho dễ dàng, tạo thêm 1 thư mục Output bên trong thư mục BlackBerry để tránh việc 2 tệp OS.signed "đánh nhau", nếu bạn không thích thì xóa OS.signed cũ đi cho lẹ;)

  1. Quay lại cửa sổ terminal.
  2. Gõ lệnh theo cấu trúc sau (CHƯA NHẤN ENTER):
bb10mt.exe pack -c "đường_dẫn\output\Patched_OS.signed" --input
  1. Sau --input, chưa nhấn Enter. Mở thư mục chứa các file đã unpack.
  2. Kéo thả 4 file sau vào terminal:
    • File .ufs
    • File .ifs
    • File .mbr
    • File .rcfs
  3. Thêm dấu phẩy (,) giữa các đường dẫn file. Bỏ qua các file .sig và .sig2.

Lệnh hoàn chỉnh sẽ trông như sau (NẾU BẠN TẠO THƯ MỤC Output trong BlackBerry):

bb10mt.exe pack -c C:\BlackBerry\Output\OS.signed --input C:\BlackBerry\unpacked\OS.ufs,C:\BlackBerry\unpacked\OS.ifs,C:\BlackBerry\unpacked\OS.mbr,C:\BlackBerry\unpacked\OS.rcfs

Lệnh hoàn chỉnh sẽ trông như sau (NẾU BẠN KHÔNG TẠO THƯ MỤC Output trong BlackBerry):

bb10mt.exe pack -c C:\BlackBerry\OS.signed --input C:\BlackBerry\unpacked\OS.ufs,C:\BlackBerry\unpacked\OS.ifs,C:\BlackBerry\unpacked\OS.mbr,C:\BlackBerry\unpacked\OS.rcfs
  1. Nhấn Enter. bb10mt sẽ tạo ra file Patched_OS.signed tại đường dẫn bạn chỉ định.
5

Tạo Autoloader Cuối Cùng

Với file OS đã vá, bước cuối là đóng gói thành một Autoloader .exe hoàn chỉnh.

  1. Mở DBBT (Darcy's BlackBerry Tools) - modified.
  2. Trong tab "Build Autoloader":
    • Chọn file OS.signed vừa tạo ở Bước 4.
    • Chọn file Radio.signed gốc từ Bước 1 (nếu thiết bị yêu cầu).
  3. Nhập tên cho file Autoloader mới.
  4. Nhấn "Build it!". DBBT sẽ tạo ra file Autoloader .exe trong cùng thư mục.
Hoàn Tất! Bạn đã có một Autoloader tùy chỉnh, sẵn sàng để flash vào thiết bị.

Flash Vào Thiết Bị

Backup Trước Khi Flash: Hãy sao lưu toàn bộ dữ liệu quan trọng trước khi tiến hành. Quá trình flash sẽ xóa sạch thiết bị.
  1. Tắt thiết bị BlackBerry.
  2. Chạy file Autoloader .exe vừa tạo.
  3. Một cửa sổ command sẽ xuất hiện – lúc này mới cắm USB vào máy tính.
  4. Đèn LED của thiết bị sẽ chuyển sang màu xanh lá – đây là dấu hiệu quá trình flash đang diễn ra.
  5. Sau khi hoàn tất, thiết bị sẽ tự khởi động lại và vào thẳng màn hình chính (home screen), bỏ qua hoàn toàn initial setup.
Chúc Mừng! Bạn đã hạ cấp thành công thiết bị BlackBerry của mình. Thiết bị sẽ khởi động thẳng vào home screen mà không cần trải qua màn hình cài đặt ban đầu nữa.
