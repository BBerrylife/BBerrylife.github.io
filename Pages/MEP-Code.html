Hướng Dẫn Unlock MEP Code Cho BBOS

⚠️ Lưu ý quan trọng: Hướng dẫn này chỉ áp dụng cho BBOS (BlackBerry OS 7.1 và cũ hơn), KHÔNG áp dụng cho BB10. Việc unlock MEP codes sẽ giúp bạn mở khóa mạng để sử dụng SIM từ các nhà mạng khác nhau.

📋 Chuẩn Bị

  • BlackBerry chạy BBOS (Bold 9900, Curve 9360, Torch 9800, v.v.)
  • Giấy và bút để ghi chép thông tin
  • Kết nối internet
  • Kiên nhẫn và làm theo hướng dẫn từng bước

🔐 Phương Pháp 1: Tìm MEP Thủ Công (Khuyên Dùng)

Bước 1: Tháo pin và tìm MEP

Tắt máy BlackBerry, tháo nắp lưng và pin ra. Kiểm tra nhãn dán phía sau pin hoặc trong khoang pin. Bạn sẽ tìm thấy:

  • PRD: Mã sản phẩm (ví dụ: PRD-12345-678)

Ghi lại thông tin này.

Bước 2: Check PRD để lấy MEP

Truy cập link sau để nhập PRD và nhận MEP:

🔗 Check PRD → MEP

Nhập mã PRD vào công cụ và bạn sẽ nhận được mã MEP tương ứng.

Bước 3: Lấy IMEI từ BlackBerry

Trên BlackBerry của bạn, gõ: *#06#

Hoặc vào: Options → About → Device and Status Information

Ghi lại số IMEI (15 chữ số).

Bước 4: Lấy MEP Code để Unlock

Sau khi có mã MEP, truy cập link sau để lấy MEP Code (unlock code):

🔓 Lấy MEP Code (Unlock Code)

Nhập mã MEP & IMEI vào công cụ và bạn sẽ nhận được các mã unlock.

🔍 Phương Pháp 2: Tìm MEP Tự Động

📌 Khi nào dùng phương pháp này?

  • Pin không tháo được (mặc dù BBOS nào cũng tháo được pin)
  • Nhãn dán bị mờ hoặc bong tróc
  • Không tìm thấy thông tin PRD trên nhãn

Bước 1: Tìm MEP

Truy cập link sau và làm theo hướng dẫn (tiếng Anh) để tìm MEP:

🔍 Link tìm MEP

Làm theo hướng dẫn trong link và bạn sẽ nhận được mã MEP.

Bước 2: Lấy MEP Code để Unlock

Sau khi có mã MEP từ Bước 1, truy cập link sau để lấy MEP Code:

🔓 Lấy MEP Code (Unlock Code)

Nhập mã MEP và lấy các mã unlock tương ứng.

⌨️ Cách Nhập MEP Code Để Unlock

Sau khi có MEP Code, làm theo các bước sau:

  1. Tắt BlackBerry
  2. Lắp SIM từ nhà mạng khác (không phải nhà mạng hiện tại đang bị lock)
  3. Bật máy lên
  4. Màn hình sẽ hiện "Enter Network MEP Code"
  5. Nhập MEP Code 1 (Network Lock) mà bạn đã lấy được
  6. Nhấn Enter
  7. Nếu thành công, màn hình sẽ hiện "Network Unlocked"

⚠️ Lưu ý:

  • Bạn chỉ có 10 lần thử. Sau 10 lần nhập sai, máy sẽ bị khóa vĩnh viễn!
  • Kiểm tra kỹ MEP Code trước khi nhập
  • Thông thường chỉ cần nhập MEP Code 1 (Network Lock)
  • Nếu MEP Code 1 không hoạt động, thử MEP Code 2, 3, 4 theo thứ tự

❓ Các Lỗi Thường Gặp

Không tìm thấy MEP hoặc PRD trên nhãn?

Sử dụng Phương Pháp 2 để tìm MEP.

MEP Code không đúng?

Kiểm tra lại mã MEP bạn đã nhập vào công cụ. Đảm bảo nhập đúng 100%, không có khoảng trắng thừa.

Đã nhập sai 10 lần, máy bị khóa?

Rất tiếc, bạn cần phải mang đến trung tâm dịch vụ chuyên nghiệp để unlock bằng phần cứng (hardware unlock). Đây là lý do tại sao bạn phải cẩn thận và kiểm tra kỹ trước khi nhập.
